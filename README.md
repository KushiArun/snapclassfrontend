# 📌 About the Project

SnapClass Landing Page is designed to introduce users to the **SnapClass AI Attendance System** through an attractive, responsive, and informative interface.

It provides visitors with an overview of the project, highlights its key features, demonstrates the user workflow, and directs users to the live AI attendance application.

# ⚙️ Installation

## Clone the Repository

```bash
git clone https://github.com/KushiArun/snapclassfrontend.git

cd snapclassfrontend
```

## Create a Virtual Environment

### Windows

```bash
python -m venv .venv

.venv\Scripts\activate
```

### macOS / Linux

```bash
python3 -m venv .venv

source .venv/bin/activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run the Application

```bash
python app.py
```

Open your browser:

```
http://127.0.0.1:5000
```
